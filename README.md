# karolinazarzyczny.github.io

This repository contains the files for my GitHub Pages website:

- `index.html` — homepage / landing page  
- `about.html`, `services.html`, `contact.html` — site pages  
- `css/styles.css` — styling  
- `images/` — headshot and sample images  
- `assets/` — optional CV or other downloadable files

How to update
1. Edit pages directly on GitHub (click a file → pencil icon → commit changes), or
2. Clone the repo locally, edit files, then `git push` to `main`.

Notes
- The site is published at: https://karolinazarzyczny.github.io
- Replace `images/photo.jpg` with a professional headshot and add sample images in `images/`.
- Replace `assets/cv.pdf` with your CV if you want the Download CV link to work.

Contact
- For small site changes, edits can be made directly in this repo.
